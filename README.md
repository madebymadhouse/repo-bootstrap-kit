# repo-bootstrap-kit

Bootstrap templates for standard Mad House repository contracts.

## Included

- `templates/README.template.md`
- `templates/CONTRIBUTING.template.md`
- `templates/.github/workflows/pr-check.yml`

## Purpose

Use this kit when creating or modernizing repos so every project starts with the same baseline for contribution quality, PR hygiene, and documentation clarity.

## Next build targets

- Add an initializer script that applies templates into a target repo.
- Add optional language packs (Node, Python, Docs-only).
- Add token replacement for project name, owner, and default branch.
